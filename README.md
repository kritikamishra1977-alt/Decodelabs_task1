Data Cleaning & Preparation using Microsoft Excel

Overview

This project was completed as part of the DecodeLabs Data Analytics Internship. The objective was to clean and prepare a raw dataset using Microsoft Excel so that it could be used for accurate analysis. The dataset contained missing values, duplicate records, and inconsistent formatting, which were identified and corrected during the cleaning process.

Objectives

- Identify missing or null values
- Remove duplicate records
- Correct inconsistent date, text, and number formats
- Improve the overall quality and consistency of the dataset
- Prepare a clean dataset for further analysis

Tools Used

- Microsoft Excel

Work Performed

- Examined the dataset to identify data quality issues.
- Handled missing values wherever required.
- Removed duplicate entries to maintain unique records.
- Corrected inconsistent date, text, and numerical formats.
- Standardized the dataset to ensure consistency.
- Verified the cleaned data to improve its reliability and accuracy.

Skills Gained

Working on this project helped me understand the importance of data cleaning before starting any analysis. I gained practical experience in identifying missing values, removing duplicate records, and organizing raw data into a structured format. I also improved my Microsoft Excel skills by applying different data-cleaning techniques and learned how even small errors in a dataset can affect the accuracy of analysis. This project strengthened my attention to detail, logical thinking, and confidence in working with real-world datasets.

Outcome

The dataset was successfully cleaned and prepared for future analysis. After removing inconsistencies and improving data quality, the final dataset became more accurate, organized, and reliable for analytical tasks.

Conclusion
This project provided valuable hands-on experience in data cleaning and preparation. It enhanced my understanding of how high-quality data forms the foundation of meaningful analysis and better decision-making.
